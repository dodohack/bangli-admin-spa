# This is the repository of admin.huluwa.uk. #

admin.huluwa.uk is the admin backend for users to manage products, orders, cms, users etc.

This backend app is based on Angular2 and Bootstrap4, it retrives business related data
from api.huluwa.uk, authenticate users from auth.bangli.uk using JWT(JSON Web Token).

### How to run this server ###

1. npm install
Install all dependences

2. npm start
Compile typescript into javascript, run the server and monitors files changes.
(see package.json for more details)
