自动优惠过滤条件

一行一组过滤条件

按照内容过滤举例

  • ^free delivery - 过滤掉以"free delivery"打头的优惠条目
  • free delivery - 过滤掉包含"free delivery"的优惠条目
  • free gift$ - 过滤掉以"free gift"结尾的优惠条目
  • TODO: more MySQL regexp

按照优惠所属商家过滤

  • AWIN|1234|Merchant name - 过滤掉Affiliate Window平台商家ID为1234的所有优惠
  • LINKSHARE|1234|Merchant name - 过滤掉Linkshare平台商家ID为1234的所有优惠
  • WEBGAIN|1234|Merchant name - 过滤掉Webgain平台商家ID为1234的所有优惠

* 商家名字为可选，为方便后期维护，建议填写商家名字

自动补齐优惠内容

  • ^[0-9]{2}% off.$|START|全场折扣:|END|散花! - 匹配内容仅为'15% off.'的优惠，把优惠内容变为'全场折扣：15% off.散花!'， 这里'START'后面的文字会自动添加到前部，'END'后面的文字会自动添加到后面。
